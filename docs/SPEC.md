# SPEC

## Goal
A kitchen management service that tracks inventory, recipes, and meal planning. Built with Kotlin and Spring Boot following hexagonal architecture.

## Scope
- Inventory management: track ingredients and quantities
- Recipe management: store and search recipes by available ingredients
- Meal planning: schedule meals based on inventory and preferences
- REST API for external integrations
- Non-goals: mobile apps, real-time notifications, ML-based recommendations
